## Installation

Clone respo

```bash
git clone https://github.com/SathiraSriSathsara/cod4x-linux-server.git
```

Run Script

```bash
cd cod4x-linux-server
sudo apt-get install dos2unix
dos2unix install.sh
sudo chmod 777 *
sudo ./install.sh
```

