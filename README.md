# Cod4x-linux-server Install Script (Ubuntu) - SaM

Bash script for Call Of Duty 4 ModernWarfia Linux server


## Installation

Clone respo

```bash
git clone https://github.com/SathiraSriSathsara/cod4x-linux-server.git
```

Run Script

```bash
cd cod4x-linux-server
chmod 777 *
./install.sh
```
